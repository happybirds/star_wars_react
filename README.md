# star-war-react
Using swapi.dev API to display information about Star Wars 

## Technologies
* React Redux & Hooks
* React router - Lazy Loading 
* Material-UI

## Running
```bash
git git@github.com:happybirds/star_wars_react.git
npm install
npm start
```


## Functional Requirements
- [x] Display people 
- [x] Display planets 
- [x] Display movies
